Capstone Project: Creating Customized Packaging Using Parametric Design Processes

2020

Package + Parametric Design
This capstone project is centered on research around parametric design and how it can elevate a brand by innovating systems of mass-customization for packaging. Parametric design, mass-customization, branding, and consumerism all offer opportunities for innovation and change. During my research, I identified that change is a common thread that is found across each of these elements of design. I also identified an opportunity to bring these components together and integrate change across and between them to create a new model of package design.

+parametric design, 3D modeling, 3D rendering, animation
Duration: 9 months

Awarded


2021

GDUSA: American Package Design Awards - Student Division

2020

Creative Communication Awards - Student Division

FORMAL PACKAGE DESIGN

This is a speculative, process-driven project that will set a precedent that could be used by any brand to exemplify change and uniqueness in their packaged products. To write the parametric scripts, data was collected from three different sources: the consumer, the product, and the brand identity.
