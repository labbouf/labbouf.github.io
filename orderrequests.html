Automated Order Requests

2023

Digital Product Design
The Order Requests feature enables customers to submit, modify, or track service orders conveniently through the portal, eliminating the need for direct contact with Sales Support. The project aims to enhance functionality within the enterprise portal by optimizing order status tracking, customer-specific data management, and improving the overall service request experience. Real-time visibility into order requests will be provided, along with automation to simplify the addition of services outlined in customers' Statements of Work (SOW) within the portal. Scalability for customers with 10+ services per location is a priority, ensuring comprehensive visibility into orders and Serviceability Results. Integration with other Service Delivery features will streamline the entire process.

+journey mapping, user flows & wireframes, prototyping & interaction design, inclusive & accessible design, business objective alignment, high-fidelity, visual design, user interface
Duration: 8 months

New Order Request

Establishing a user-friendly, step-by-step process for submitting order requests, facilitating seamless provision of necessary information for system processing and Sales Support technician assistance. This ensures accurate data provision and timely quotation delivery to customers.

Request Tracker

Comprehensive order tracking including customer-provided details, milestone updates, and integration with Network Availability Prediction (NAP) program for Serviceability Results. Seamless guidance on next steps and connection to subsequent Service Delivery features upon request completion, ensuring a smooth process for customers within the portal.

Service Selection

Streamlining service selection for customers by automatically populating their Statement of Work (SOW) services into a dropdown menu. This prevents duplication by enabling users to select each service only once. Additionally, introducing account-level templates to standardize settings across all users for enhanced efficiency when submitting a new order.

Multi-Site Order Request Details

Extending full visibility to bulk Order Requests by displaying individual sites within a single order. Enabled by the backend system categorizing them as multiple sites under one MES ticket in the Network Availability Prediction program.
